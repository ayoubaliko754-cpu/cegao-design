🚀 第三步：部署到 Vercel
─────────────────────────────────────────────────────────────
[ ] 登录 Vercel
[ ] 点击 "Add New" → "Project"
[ ] 找到并选择 GitHub 仓库
[ ] 确认配置（Next.js 自动检测）
[ ] 点击 "Deploy" 按钮
[ ] 等待部署完成（2-5 分钟）
[ ] 确认显示 "Deployment successful"
[ ] 点击临时地址测试访问
[ ] 记录临时地址：https://cegao-design.vercel.app

🔗 第四步：绑定阿里云域名
─────────────────────────────────────────────────────────────
[ ] 在 Vercel 进入项目 Settings
[ ] 点击 "Domains" 标签
[ ] 在 "Add domain" 输入您的域名
[ ] 点击 "Add" 按钮
[ ] 记录 DNS 记录信息：
    - A 记录值：76.76.21.21
    - CNAME 记录值：cname.vercel-dns.com
[ ] 复制这些信息

⚙️ 第五步：配置阿里云 DNS
─────────────────────────────────────────────────────────────
[ ] 登录阿里云 DNS 控制台
[ ] 找到您的域名
[ ] 点击 "解析设置"
[ ] 添加 A 记录：
    [ ] 记录类型：A
    [ ] 主机记录：@
    [ ] 记录值：76.76.21.21
    [ ] TTL：600
    [ ] 点击 "确认"
[ ] 添加 CNAME 记录：
    [ ] 记录类型：CNAME
    [ ] 主机记录：www
    [ ] 记录值：cname.vercel-dns.com
    [ ] TTL：600
    [ ] 点击 "确认"

⏳ 第六步：等待 DNS 生效
─────────────────────────────────────────────────────────────
[ ] 返回 Vercel 的 Domains 页面
[ ] 点击 "Verify" 按钮
[ ] 等待域名状态变化
    [ ] 状态变为 "Valid Configuration" = 成功
    [ ] 状态仍为 "Pending" = 继续等待
[ ] 等待时间：5-30 分钟（最长 24 小时）
