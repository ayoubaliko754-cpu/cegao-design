# 策高室内装饰设计 - 网站使用说明

## 项目概述

这是一个现代化的企业官网，采用 Next.js 16 + Tailwind CSS 4 + shadcn/ui 技术栈开发，具有响应式设计，完美适配桌面端和移动端。

## 技术栈

- **框架**: Next.js 16 (App Router)
- **语言**: TypeScript 5
- **样式**: Tailwind CSS 4
- **UI 组件**: shadcn/ui
- **包管理器**: pnpm

## 目录结构

```
src/
├── app/
│   ├── globals.css          # 全局样式
│   ├── layout.tsx           # 根布局
│   └── page.tsx             # 首页
├── components/
│   └── ui/                  # shadcn/ui 组件库
├── data/
│   └── site-content.ts      # 网站内容配置文件（重要！）
└── lib/
    └── utils.ts             # 工具函数
```

### 方法 2：添加项目图片

1. 在 `public/images/` 目录下放置您的图片文件
2. 在 `site-content.ts` 中更新对应的图片路径

#### 创建图片目录
```bash
mkdir -p public/images
```

#### 建议的图片尺寸
- Hero 背景图: 1920x1080px
- 项目案例图: 800x600px
- 新闻图片: 600x400px
- 关于我们图片: 1200x800px

### 方法 3：修改颜色主题

在 `src/app/globals.css` 文件中修改 CSS 变量：

```css
:root {
  --primary: oklch(0.45 0.12 250);      /* 主色调 */
  --secondary: oklch(0.96 0.01 250);    /* 次要色调 */
  --background: oklch(0.99 0.005 95);   /* 背景色 */
  --foreground: oklch(0.15 0.015 260);  /* 文字颜色 */
  /* ... 其他颜色变量 */
}
```

## 开发命令

### 启动开发服务器
```bash

```
服务将在 http://localhost:5000 运行，支持热更新。

### 构建生产版本
```bash
coze build
```

### 启动生产服务器
```bash
coze start
```

### 安装依赖
```bash
pnpm install
```

### 安装新依赖
```bash
pnpm add <package-name>
```

## 响应式设计说明

网站已针对不同设备进行优化：

- **桌面端**: ≥ 1024px (lg 断点)
- **平板端**: 768px - 1023px (md 断点)
- **移动端**: < 768px

在移动端：
- 导航栏会自动折叠为汉堡菜单
- 网格布局会自动调整为单列
- 字体大小会相应调整

## 常见问题

### Q: 修改内容后没有立即生效？
A: 开发服务器支持热更新，如果未生效，请尝试刷新浏览器（Ctrl+F5 或 Cmd+Shift+R）。

### Q: 如何添加新的项目案例？
A: 在 `src/data/site-content.ts` 的 `projects.items` 数组中添加新对象即可。

### Q: 如何修改网站颜色？
A: 编辑 `src/app/globals.css` 文件中的 CSS 变量。

### Q: 图片显示不出来？
A: 请确保图片已放置在 `public/images/` 目录下，且路径配置正确。
