# 上传代码到 GitHub 的两种方法

## 方法 1：通过 GitHub 网页上传（最简单）

### 步骤：

#### 1. 创建 GitHub 仓库
```
1. 访问：https://github.com/new
2. 填写：
   - Repository name: cegao-design
   - Description: 策高室内装饰设计官网
   - 选择 "Public"
3. 点击 "Create repository"
```

#### 2. 上传文件
```
1. 在新仓库页面
2. 找到 "uploading an existing file" 链接
3. 点击进入上传页面
4. 拖拽以下文件和文件夹：

需要上传的文件：
- src/ 文件夹
- public/ 文件夹（如果有）
- package.json
- next.config.ts
- .gitignore
- tsconfig.json
- tailwind.config.ts
- .babelrc
- .npmrc
- .coze
- README.md
- 其他 .md 文档文件

不需要上传的文件（在 .gitignore 中）：
- node_modules/
- .next/
- .vercel/
- .coze-logs/
```

#### 3. 提交
```
1. 填写提交信息：
   Commit message: Initial commit
2. 点击 "Commit changes"
3. 等待上传完成
```

---

## 方法 2：使用命令行推送（需要技术能力）

### 步骤：

#### 1. 在服务器上执行
```bash
cd /workspace/projects
git add .
git commit -m "Initial commit"
```

#### 2. 添加远程仓库
```bash
git remote add origin https://github.com/您的用户名/cegao-design.git
```

#### 3. 推送
```bash
git branch -M main
git push -u origin main
```
