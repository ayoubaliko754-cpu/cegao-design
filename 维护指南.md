## 📋 三种维护方式（按简单程度排序）

### 方式 1：通过网页管理后台（推荐 ⭐⭐⭐⭐⭐）

**访问地址：**
```
http://9.129.139.119:5000/admin
```

**操作步骤：**

#### 第一步：访问管理页面
1. 打开浏览器
2. 在地址栏输入：`http://9.129.139.119:5000/admin`
3. 按回车键

#### 第二步：添加新项目
在管理页面左侧填写信息：

| 字段 | 说明 | 示例 | 必填 |
|------|------|------|------|
| 项目名称 | 案例的名称 | "湖畔别墅" | ✅ 是 |
| 项目分类 | 案例类型 | "住宅设计" | ✅ 是 |
| 项目描述 | 详细描述案例特点 | "现代简约风格..." | ✅ 是 |
| 项目面积 | 项目总面积 | "320㎡" | ❌ 否 |
| 项目位置 | 项目所在城市 | "杭州" | ❌ 否 |
| 项目图片 | 案例展示图 | 点击上传按钮 | ❌ 否 |

#### 第三步：上传图片
1. 点击"选择文件"按钮
2. 选择您的图片文件
3. 建议尺寸：800x600px
4. 支持格式：JPG、PNG

#### 第四步：保存
1. 点击"添加项目"按钮
2. 等待保存完成
3. 项目会立即显示在首页

**优点：**
- ✅ 像填写表单一样简单
- ✅ 不需要懂任何技术
- ✅ 随时可以更新
- ✅ 支持图片上传

## 🔧 常见问题

### Q1: 图片上传失败怎么办？

**可能原因：**
- 图片文件太大（超过5MB）
- 图片格式不支持
- 网络问题

**解决方案：**
1. 压缩图片（使用在线工具）
2. 转换为JPG或PNG格式
3. 检查网络连接

---

### Q2: 如何删除已有的项目？

**方法 1：使用管理页面**
1. 访问管理页面

3. 点击垃圾桶图标
4. 确认删除

告诉我删除哪个项目，我帮您操作。
