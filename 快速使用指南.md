## 🎨 添加案例图片详细指南

### 准备工作

**图片要求：**
- 推荐尺寸：800x600px（4:3比例）
- 文件大小：建议小于2MB
- 支持格式：JPG、PNG
- 建议：使用清晰的高质量图片

**图片处理工具（如需调整）：**
- 在线工具：https://www.iloveimg.com/resize-image
- 电脑自带：画图工具（Windows）、预览（Mac）

### 添加步骤

#### 方法 1：使用管理后台（最简单）
```
1. 访问：http://9.129.139.119:5000/admin
2. 填写项目信息
   - 项目名称：如"湖畔别墅"
   - 项目分类：如"住宅设计"
   - 项目描述：详细描述项目特色
   - 项目面积：如"320㎡"
   - 项目位置：如"杭州"
3. 点击"选择文件"上传图片
4. 点击"添加项目"
5. 完成！立即生效
```

#### 方法 2：让我帮您
```
1. 准备好图片和项目信息
2. 告诉我：
   - 项目名称
   - 项目分类
   - 项目描述
   - 项目面积
   - 项目位置
   - 图片文件
3. 我帮您上传和配置
```

## 🔧 常见问题

### Q1: 如何访问管理后台？
A: 访问 http://9.129.139.119:5000/admin

### Q2: 图片上传失败怎么办？
A:
- 检查图片大小（建议小于2MB）
- 检查图片格式（支持JPG、PNG）
- 压缩图片后重试

### Q3: 如何删除项目？

### Q4: 修改后多久生效？
A: 使用管理后台修改，立即生效（1-2秒内）

### Q5: 能否批量上传？
A: 当前版本支持逐个添加。如需批量上传，可以一次性发给我，我帮您处理。
