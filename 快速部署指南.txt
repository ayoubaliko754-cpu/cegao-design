【部署准备】
─────────────────────────────────────────────────────────────
✅ 阿里云域名（已备案）
✅ 网站代码（已准备好）

【第一步：注册 Vercel】
─────────────────────────────────────────────────────────────
1. 访问：https://vercel.com/signup
2. 使用 GitHub 或邮箱注册
3. 完成注册

【第二步：导入项目】
─────────────────────────────────────────────────────────────
1. 登录 Vercel
2. 点击 "Add New" → "Project"
3. 导入 GitHub 仓库
4. 点击 "Deploy"
5. 等待 2-5 分钟

【第三步：绑定域名】
─────────────────────────────────────────────────────────────
1. 在 Vercel 项目中，点击 "Settings"
2. 点击 "Domains"
3. 添加您的域名
4. 复制 DNS 记录

【第四步：配置 DNS】
─────────────────────────────────────────────────────────────
登录阿里云 DNS 控制台
https://dns.console.aliyun.com/

添加两条记录：

记录1（A 记录）：
- 类型：A
- 主机：@
- 值：76.76.21.21

记录2（CNAME 记录）：
- 类型：CNAME
- 主机：www
- 值：cname.vercel-dns.com

【第五步：等待生效】
─────────────────────────────────────────────────────────────
1. 返回 Vercel
2. 点击 "Verify"
3. 等待 5-30 分钟
4. 确认显示 "Valid Configuration"

【第六步：测试访问】
─────────────────────────────────────────────────────────────
在浏览器输入：https://您的域名.com

看到网站 = 成功！✅
